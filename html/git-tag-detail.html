git tag

图解   回到主页

  • git tag -a Vxx -m "XXX"

    • 在本地仓库创建一个标签，其名称为Vxx，描述信息为XXX（-a: --annotate, -m: --message）

  • git tag -a Vxx -m "XXX" <commit-id>

    • 在本地仓库的commit-id节点创建一个标签，其名称为Vxx，描述信息为XXX（-a: --annotate, -m: --message）

  • git tag -l

  • git show-ref --tag

    • 查看本地仓库的所有标签（-l: --list）

  • git show Vxx

    • 查看本地仓库Vxx标签的详细信息

  • git tag -d Vxx

    • 在本地仓库删除一个名称为Vxx的标签 （-d: --delete）

  • git push origin Vxx

    • 推送本地仓库的Vxx标签到远程仓库

  • git push origin :Vxx

    • 删除远程仓库的Vxx tag
