git fetch


抓取远程代码库内容到本地代码库。请参考：一张图了解git

  • git fetch / git fetch origin

    • 抓取远程代码库所有分支的内容到本地代码库。

  • git fetch origin <branch>

    • 抓取远程代码库branch分支的内容到本地代码库，branch分支必须存在。

    $ git fetch origin abc
    fatal: Couldn't find remote ref abc

    $ git fetch origin master
    From https://github.com/yang-an-china/easygit
    * branch master -> FETCH_HEAD


    $ git fetch origin develop
    From ssh://gerritro.***.com.cn:29418/###
    * branch develop -> FETCH_HEAD
    * [new branch] develop -> origin/develop

  • git fetch repository <branch>

    • 抓取远程代码库branch分支的内容到本地代码库，branch分支必须存在。

    $ git fetch https://github.com/yang-an-china/easygit.git master
    From https://github.com/yang-an-china/easygit
    * branch master -> FETCH_HEAD
    $ git lg FETCH_HEAD -2
    * 7c36001 - (HEAD -> master, origin/master, origin/HEAD) + clone fetch refs (2 days ago) <yang-an-china>
    * fd91629 - +HEAD add commit diff reset status index working-tree (5 days ago) <yang-an-china>
