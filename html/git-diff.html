git diff


比较不同提交之间，提交与工作树之间，不同分支之间的差异。

  • git diff

    • 工作树跟本地缓存的差异。（即在工作树中尚未add）

  • git diff --cached/staged

    • 本地缓存跟本地仓库的差异。（即在缓存中尚未commit）

  • git diff HEAD

    • 工作树跟本地代码库的差异。

  • git diff commitA commitB

    • commitA跟commitB的差异。

  • git diff branchA branchB / git diff branchA..branchB

    • branchA分支跟branchB分支的差异。

  • git diff branchA...branchB

    • branchA分支从branchB分支上分离之后的差异。

  • git diff --diff-filter=MRC

    • 差异过滤。（M -> modification，R -> rename，C -> copy，A -> addition，D -> deletion，T -> type，U -> unmerged）
