git revert


回退本地代码库到某一个commit（注意：包含commit本身）。

  • git revert HEAD

    • 把HEAD回退，也就是回退本地代码库到HEAD^。

  • git revert commitA..commitB

    • 把commitA(包含)到commitB(不包含)之间的commit回退掉。

  • git revert master~4..master~2

    • 把master分支的倒数第4个(包含)到倒数第2个(不包含)之间的commit回退掉，参见 图解2。

  • git revert -n master~4..master~2

    • 把master分支的倒数第4个(包含)到倒数第2个(不包含)之间的commit回退掉，没有commit，参见 图解3。

  • git revert --continue

  • git revert --abort

  • git revert --quit
