git branch


分支相关操作（增删改查）。

  • 增

  • git branch <branch>

    • 从当前分支拷贝一个新分支，该分支名称为branch

  • git branch (--set-upstream-to=<upstream> | -u <upstream>) [<branchname>]

    • 设置upstream，请参见：upstream

  • 删

  • git branch -d <branch>

    • 删除本地<branch>分支（-d -> --delete）

  • git branch -D <branch>

    • 强制删除本地<branch>分支（-D -> --delete --force）

  • git branch -r -d(-D) <origin/branch>

    • （强制）删除本地远程跟踪<origin/branch>分支（-r -> --remote）

  • 改

  • git branch -m(-M) <branch> <new-branch>

    • （强制）把本地<branch>分支重新命名为<new-branch>（-m -> --move, -M -> --move --force）

  • 查

  • git branch / git branch -l

    • 列出所有的本地分支（-l -> --list）

  • git branch -r

    • 列出所有的本地远程跟踪分支（-r -> --remote）

    本地远程跟踪分支对应的原文是remote-tracking branch

  • git branch -a

    • 列出所有的本地分支和本地远程跟踪分支

  • git branch -vv

    • 列出所有的本地分支（包括sha1和commit）和对应的远程跟踪分支
