upstream


upstream是本地分支关联的远程分支。

  • 如何查看upstream

    • 命令：git branch -vv

      • $ git branch -vv
      * master a6eef83 [origin/master] add git log modify some style

      其中远程origin对应的master分支 [origin/master] 就是本地master分支的upstream

  • 如何设置upstream

    • 命令：git branch (--set-upstream-to=<upstream> | -u <upstream>) [<branchname>]

      • 如果branchname不填写的话，则默认是当前分支。
      • 示例：设置本地分支mybranch的远程跟踪分支为master。


        • $ git branch -vv
        mybranch a6eef83 add git log modify some style
        * master a6eef83 [origin/master] add git log modify some style

        $ git branch -u origin/master mybranch
        Branch mybranch set up to track remote branch master from origin.

        $ git branch -vv
        mybranch a6eef83 [origin/master] add git log modify some style
        * master a6eef83 [origin/master] add git log modify some style

        $ git branch --unset-upstream mybranch

        $ git branch -vv
        mybranch a6eef83 add git log modify some style
        * master a6eef83 [origin/master] add git log modify some style

        $ git branch --set-upstream-to=origin/master mybranch
        Branch mybranch set up to track remote branch master from origin.

    • 命令：git checkout <branchX>

      • 本地创建branchX分支用来跟踪远程branchX分支。
      • 示例：


        • $ git branch -vv
        * master a6eef83 [origin/master] add git log modify some style

        $ git checkout -b mybranch --track origin/master
        ...
        Branch mybranch set up to track remote branch master from origin.
        Switched to a new branch 'mybranch'

        $ git branch -vv
        * master a6eef83 [origin/master] add git log modify some style
        mybranch a6eef83 [origin/master] add git log modify some style
