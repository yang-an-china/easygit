git init


创建一个新的git仓库或者对git仓库重新初始化。

  • git init

    • 在当前目录创建一个代码仓库

  • git init --bare

    • 在当前目录创建一个裸仓库（仅包含版本管理[即.git文件夹下的内容]，不包含代码内容）

  • 如何在本地进行git练习

    • 创建一个代码仓库
      • /home/git/rep$ git init --bare
    • 创建两个用户
      • /home/git/user$ git clone /home/git/rep/ usr1
      • /home/git/user$ git clone /home/git/rep/ usr2
    • 可以在usr1和usr2下模拟两个用户进行git练习
