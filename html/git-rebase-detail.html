git rebase


git rebase的作用是变基，可以作用到分支上，也可以作用到某个分支的commit上。

  • git rebase <upstream> [branch]

    • 把[branch]分支变基到upstream(一般是分支)

    如果[branch]为空，则是当前分支

    错误用法：git rebase origin master

    正确用法：git rebase origin/master master

  • git rebase -i

    • rebase的交互模式

    一般用来处理某个分支下的commit，参见图解：git rebase i
